Useful Websites:

A collection of useful websites for passive information gathering

1. Whois.com

Whois.com will run a whois search on a domain. This can give you information such as registrant name, address, organization and email.

2. CentralOps.net

CentralOps.net will do an address lookup, DNS lookup, WHOIS lookup, traceroute, and more.

3. IPinfo.io

IPinfo.io provides information about IP addresses, including geolocation, ISP details, and hosting information.

4. Reverse Shell Generator

Has a collection of reverse shells in different languages that you can quickly create with your IP and specified port.

5. Robtex.com

Robtex.com provides a range of network-related information, including DNS records, IP addresses, and network graphs.

6. Shodan.io

Shodan.io is a search engine for Internet-connected devices. It allows users to search for devices based on criteria such as IP address, operating system, and open ports.

7. MXtoolbox.com

MXtoolbox.com provides a range of email-related tools, including MX record lookup, SPF record checker, and email blacklisting check.

8. Wappalyzer.com

Wappalyzer.com is a tool that allows users to identify the technologies used by websites, including content management systems, analytics tools, and advertising networks. There is also a browser extension that is very handy.

9. Hunter.io

A tool used to find email addresses.

10. Exif Tool Online

Exif tool allows you to upload a photo and gain its metadata.

11. Twitter Advanced Search

Allows you to search Twitter with a stronger set of categories.

12. CyberChef

CyberChef is a tool used to encode/decode text.

13. VirusTotal.com

VirusTotal is used to identify and analyze malicious files, IPs, Domains, and URLs.

14.Pentest Monkey

Has Cheat Sheets, Shells, SQL Injections.
