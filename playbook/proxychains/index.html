Proxychains

You can download Proxychains here.

Basic Facts:

  • Proxychains is a command-line tool that uses a TCP connection to allow you to run other programs through a proxy server.
  • This allows you to execute commands on a resource that would otherwise be unreachable.
  • Can be much slower than other tools - be prepared to wait longer than usual for scan results.
  • Reads instructions from a config file, so some scans where you normally add flags, with proxychains you don't have to.
  • This can be combined with using Chisel as a pivot in more complicated scenarios.
  • Can be used very easily just by putting "proxychains" in front of any command you want to use.

CTF Usage:

Proxychains will often be used in combination with another tool called Chisel. It can also be used on your browser with Foxy Proxy, or you can use Burp Suite. For example, if you need to run an nmap scan on a host you have no route to, but you have user access, you could chisel a proxy back to your Kali and by using proxychains, you could scan the remote host.

Example Scenario:

Kali is hosting a reverse chisel server on port 6464 Target is running chisel client running for socks4/5 Kali receives connection on a specific port, for example, 1080. Edit the last line of the /etc/proxychains.conf file to allow this port Now you can execute proxychains commands.

You can also set up proxychains with Foxy Proxy and Burp Suite. >
