Searchsploit

Basic Facts:

  • Searchsploit is a command-line tool that can help you search for exploits that have pre-existing code.

CTF Usage:

Once you have identified some versions of services running on the target, you can use Searchsploit to quickly search for any low-hanging fruit.

You can search by keywords.

Once you've found an exploit that looks like it fits your criteria, you can download it to your current directory with the -m flag, and it will be saved to your current directory.

Before you run the exploit, you should read through it and try to understand what it does. You also need to look out for anywhere that you need to edit the information (e.g., add your IP, port, etc.).
