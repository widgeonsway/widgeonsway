Chisel

    Basic Facts:
  • You can download Chisel here.
  • Chisel can be used for both Linux and Windows. You just need to be sure to use the right version.
  • Chisel allows you to quickly and easily create tunnels through the chisel client / server
  • Does not given you any information or do any enumeration, it just creates a tunnel between you and the target machine.

CTF Usage:

1. Download

While you may have previously downloaded Chisel, there are a lot of versions specific to certain operating systems.

2. Get Chisel onto Target Machine

You need to have Chisel on both your machine and the compromised machine.

On the compromised machine, you can use whatever file transferring tool you prefer or whatever the machine allows. This could be wget, curl, python, powershell, or any other application on the target. Don't forget to chmod +x the file.

3. Create the Connection

    Start the Chisel server on your machine.
  • ./chisel server -p <Port> --reverse
    Start the Chisel client on the target.
  • ./chisel client <Your Machines IP> R:<Port>:<Target IP>:<Target Port>

The Port is the port you choose to connect over. The Target port is which port of the target you will be compromising. Verify that the server received a connection.

4. Get ready to use another tool like Proxychains or SSH forwarding

Chisel is just a networking tool. You need another tool to enumerate or run exploits. After connecting with Chisel, you can use Proxychains to run your command.
