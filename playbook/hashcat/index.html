Hashcat

You can download Hashcat here

Basic Facts:

  • Used to brute-force passwords
  • While it is similar to John the Ripper, it is usually faster.

CTF Usage:

1. Find a Hash

2. Determine what type of hash it is

Go to this website and find the Hash-mode based off matching the style of your hash to the examples
You will call the mode in your command by using -m

3. Save the hash into a .txt file

4. Run your command

This should include the mode you found, your text file and a wordlist hashcat -m <mode> file.txt <wordlist>

5. Find your Results

Finding the result can be a bit tricky if you don't know where to look.
Hashcat appends the original hash with a colon and then the found password.
