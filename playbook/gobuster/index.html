Gobuster

When your nmap results show that there is a website, one of the first things to do is use Gobuster.

Basic Facts:

  • In a CTF challenge, Gobuster can be used to enumerate files and directories on a website. Gobuster is an open source command line tool that uses wordlists to brute-force websites, directories and files in websites, DNS subdomains, and more. It comes pre-installed on Kali, but if you are using another distro you can install it here.
  • The basic format of a gobuster command format is: "gobuster <option> -u <website.com> -w <wordlist>"

Common Scans:

  • gobuster dir -u http://website.com/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt
  • gobuster vhost -u http://webiste.com/ -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt --append-domain
  • gobuster dir -u http://website.com -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x.html,.css,.js

CTF Challenge Usage:

1. Know the target IP. In some challenges you are directly given the IP or IP's, while in others you may have to find the IP.

2. Know the command you want to use:

3. A good starting place for a CTF is to use the dir option. You can type "gobuster dir -h" to get more information on how to use this.

4. You need to specify that you will be using a URL. You can do this with -u

5. Use -w to specify which wordlist you want to use. This can be any wordlist, you can even make your own. This scan will display any directories found that match the chosen wordlist.

6. When the scan is finished, you need to know what to look for. Some CTF challenges will be obvious and only have a few results, and some will have so many directories in an attempt to try to send you into a rabbit hole of searching. Sometimes there will be directories like /login or /admin and you should always check those out. Other times there may be a directory that is somehow related to the name of the challenge. This just means that people who make CTF challenges are nerds and you should be on the lookout for any signs of wordplay.

7. You should also scan for vhosts as this is becoming an increasingly popular tactic in CTF challenges.

8. File Extensions Use -x to search for file extensions. Separate each file type with a comma and no space.

9. HTTPS To scan a https website, you can use the -k flag
