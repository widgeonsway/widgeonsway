FFUF

You can download FFUF here

Basic Facts:

  • Stands for Fuzz Faster U Fool
  • FFUF is similar to Gobuster but it is faster and more customizable.
  • FFUF can be used on Linux, Windows, and MacOS
  • FFUF is a fuzzer - it replaces parts of the target URL with different values
  • Make sure that you also have SecLists installed.

Common Scans:

  • ffuf -w <wordlist> -u <http://website.com>/FUZZ
  • ffuf -u <http://website.com>/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-files-lowercase.txt

CTF Challenge Usage:

Once you know the target and what you want to look for you can start customizing your commands. Knowing specifics like what wordlist to use will come with practice but to start out with there are a few things to know.

Filtering Status Codes

You can filter out specific HTTP status codes by adding -fc and the code you would like to omit. This will hide anything with that code from the results, making it easier to sort through.

Filtering Size

You can filter out size with -fs

Looking for Certain Extensions

Use the -e flag to look for certain extensions. For example, once you find a directory that you want to know more about, you can use the command:
ffuf -u <http://website.com>/FUZZ -w <wordlist> -e .txt You can search for more than one type of file at once, just separate with a comma and no space.

Recursive

Adding -recursion to your command will add a bit of automation to your life. If the scan finds a valid result, it will then scan that directory with the same scan that found it. If you find a page like /assets, FFUF will then run a scan /assets/FUZZ.

VHOST/Subdomain

To look for subdomains with FFUF, you add -H "HOST: FUZZ" to your command
