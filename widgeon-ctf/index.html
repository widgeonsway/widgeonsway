How to Set Up VirtualBox

  1. Download and install VirtualBox.
  2. Import the downloaded .ova file:
    • Download the .ova image from this page and import it into your virtual machine manager.
    • Click import to import the machine.
    • Create a new NAT Network.
    • Do this by going File → Preferences → Network.
    • Add a new network with this button.
    • Create the network 10.1.64.0/24.
    • Network name can be your choice, but the IPs are static.
    • Assign your new network to the target machine (HOOT, STORK, CLUCK).
    • Assign the same network to your machine.
  3. Start the VM and log in (if needed).
  4. Use the provided IP address to access the machine from your host (make sure you're in the same network).
